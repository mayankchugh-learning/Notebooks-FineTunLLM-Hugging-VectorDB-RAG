# LangChain & LLM Fine-tuning Experiments

A collection of Jupyter notebooks covering LangChain, LLM fine-tuning, 
and vector database implementations.

## Fine-tuning
- Fine_Tune_Llama2_on_Custom_data.ipynb — LLaMA2 fine-tuning on custom 
  datasets (classification and summarisation tasks)
- Finetune_LLM_using_DPO.ipynb — Fine-tuning using Direct Preference 
  Optimization (DPO)
- GGUF_Quantization.ipynb — Model quantization for efficient deployment

## Vector Databases
- FAISS_Demo.ipynb — FAISS vector similarity search
- FAIIS_Demo_mine.ipynb — Custom FAISS implementation
- ChromaDB_demo.ipynb — ChromaDB vector store

## LangChain & Integrations
- APIs.ipynb — API integrations with LangChain
- Huggingface_with_Langchain.ipynb — Hugging Face + LangChain
- Huggingface-OpenAI-Gemini.ipynb — Multi-model integrations
- Ineuron_custom_website_chatbot.ipynb — Custom RAG chatbot

## Tools Used
Python · LangChain · Hugging Face Transformers · FAISS · ChromaDB · 
Google Colab · LLaMA2 · OpenAI API
